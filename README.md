# **Blockvest Social** 🌍🚀  
**Decentralized Micro-Investing & Impact Platform**  

## **About the Project**  
_Blockvest Social_ is a **blockchain-powered social lending platform** that enables **peer-to-peer micro-investments** for small-scale entrepreneurs, artisans, and farmers. Built on **Algorand**, it ensures **transparent, trustless, and community-governed** financial support, empowering underserved communities with **affordable credit** while providing **on-chain dividends** to investors.  

## **Key Features**  
✅ **Smart Contract Lending** – Investors fund businesses via **decentralized loan agreements**, ensuring instant payouts & accountability.  
✅ **Trustless Credit Score System** – Entrepreneurs **earn reputation NFTs** based on repayment history, securing better loan terms.  
✅ **Gamified Social Impact** – Investors receive **NFT badges & rewards** for funding impactful businesses.  
✅ **Community-Governed Finance** – Lenders **vote on lending rules**, creating truly **decentralized financial inclusion**.  
✅ **Marketplace for Sustainable Growth** – Businesses **sell products/services**, allowing investors to **redeem earnings as goods** instead of cash withdrawals.  

## **Tech Stack**  
- **Blockchain:** Algorand Smart Contracts  
- **Frontend:** Next.js, React, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Storage:** IPFS for decentralized asset handling  
- **Wallet Integration:** Algorand blockchain transactions  

## **Getting Started**  
1️⃣ Clone the repository:  
```bash
git clone https://github.com/your-repo/blockvest-social.git
```  
2️⃣ Install dependencies:  
```bash
npm install
```  
3️⃣ Start the development server:  
```bash
npm run dev
```  
4️⃣ Connect to an Algorand wallet to begin investing or lending!
